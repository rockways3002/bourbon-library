# Rakbuku
	- An example for Bourbon, Neat, Bitters, Refills
# Install
	** Your system
		- Install nodejs
		- Install npm
		- Install bower
		- Intsall grunt-cli
	** Example 
		- https://github.com/rockways3002/bourbon-library.git
		- Checkout develop branch
		- Run 'sudo npm install'
		- Run 'bower install'
# Run site
	- Run 'grunt serve'
	- Open browser and run with 'locahost:9000'
