# Rakbuku
	- An example for Bourbon, Neat, Bitters, Refills
# Install
	** Your system
		- Install nodejs
		- Install npm
		- Install bower
		- Intsall grunt-cli
	** Example 
		- Clone sourecode: https://github.com/nhanhv/Rakbuku.git
		- Run 'cd bourbonexample' and  checkout develop branch
		- Run 'sudo npm install'
		- Run 'bower install'
# Run site
	- Run 'grunt serve'
	- Open browser and run with '192.168.33.25:9000'
